# Aleternative assignment 2

This project applies learning analytics techniques to explore how student preparation influences academic performance and consistency.

## Objectives
- Analyze relationships between subject performance
- Evaluate the impact of preparation on outcomes
- Identify at-risk student groups

## Visualisations Used
- Heatmap (correlation analysis)
- Box Plot (performance variability)
- Radar Chart (behavioral profiles)
- Sankey Diagram (flow to outcomes)

## Dataset
Students Performance in Exams (Kaggle)  
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

## Key Insight
Preparation improves performance and reduces variability, making it a strong predictor of student success.
